Add Internal Mail

  The system enables to you to add internal document electronic or paper  through the Add screen from Internal Mail

  User should when add an internal document fill all the mandaotry fields in document properties

  In case user doesn't fill one of mandatory fields , the fields will appears in red color 

  The type of the document is determined elctronic or paper when move the toggle of document type 

  User can manage the following data when add a new internal document :

  Add tags to the document for easy acess when click on Manage document tags

  The ability to add linked documents by search for any document in search for document screen from Manage Linked Documents

  Add an attachments by importing or drag and drop the attachments or by Scanner

  The ability to add comments from Manage document comments 

  The ability to add an external employee or new entity from Manage Linked entities and fill all the mandatory fields 

  Notes :

In case user add an internal electronic document and select the document type "Departmental" the document will be approved then move in Approved screen

In case user add an internal document and select the document type "Personal" the document will be approved  from user inbox then send to the user that will be selected from Send screen

  In case add an internal document without content , the document will be saved in Prepare screen

  In case you don't enter any of data , it will appears this sign

  In case you enter all the data , it will appears this sign

  In case you add an electronic internal document , user can save it as draft in Draft screen when click on "Save As Draft" button
