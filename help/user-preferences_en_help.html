User Preferences

  User Preferences allow you to customize your system settings , edit all your personal data and manage your forwarded mail by viewing it on the screen .

  The User Preferences screens is divided into several tabs :

   General : you can edit your personal data then click on save button

   Notification Settings : to control Notifications alerts sent by you :

   Activate Subscription SMS so that you can be notified on your phone with the document sent to you

  Activate Subscription Email so that you can be notified on you Email with the document sent to you

  Out Of Office Settings : select a user's assistant in case the user will be out of office or not out of office by moving the toggle and then select the user on behalf

                                                      

     Notes :

        When select a delegated user the system will send the document to his user inbox .

         When a user adds you as a delegator the Proxy Mail appear to you inbox queue .

    User Comments : user can add a personal comments that are always used .

     Workflow Groups : the ability to add a private workflow groups to the user when click on Add icon then enter the basic info and add the group members.

   Folders : create a private folders by pressing on "Add Root Folder"  button then fill the mandatory fields then click on add button

Signature : create s specific signature to the user by uploading a signature in "png format to be placed as a signature to the user
