Rejected screen - Outgoing Mail

Tawasol system enables to users from Deleted screen to view the document that have been reject but still not take a decision yet about Deleted when click on Deleted action from Review screen

The rejected documents appear in the Deleted screen

User can take any of the following actions in the Deleted screen from the step processor :

View Tracking Sheet : user can view reports about everything happens in the document

Remove Permanently :By selecting this action the book will be removed from the program

Open or Preview document : the ability to open the Deleted document and view all the general data when click on "Open" or "Preview"

Document Information : the system displays all the information which have been entered when add the document when click on document information

Archive : archive the Deleted document

Notes :

There are some actions appear or disappear in Rejected screen in accordance with the permissions granted to the user
