Add Outgoing Mail

The system allows to you to add an outgoing document from Outgoing Mail

You should first fill the mandatory fields in document properties when add the outgoing document

In case you add an outgoing document , user can add a composite document and send the document for more than one correspondence site when enables the check box "Copy for each correspondence site"

When the user approve the composite document from User Inbox a confirmation message appears as shown in the following image :

                                                                                  

When Manage Correpsondence sites the system allows to you a distribution lists which have been added from admin and every distribution list contains more than one correspondence sittes and the user can add more than one correspondence sites from the distribution list selected

                                                                          

The ability to manage the following data when add an outgoing document :

The ability to add tags from Manage Document Tags by entering the name of the tag then press enter from the keyboard

The ability to add linked documents by search for any document in search for document screen from Manage linked Documents

Add an attachments by importing the attachments or by drag and drop from Manage Attachment documents

Add comments for document from Manage Document Comments

The ability to add an external employee or new entity from Manage Linked entities and fill all the mandatory fields then click on save button

     Notes :

The outgoing document can not be added if one of mandatory field left as blank .

User can add an outgoing document electronic or paper when click on Document Type icon at the top right of the screen .

The outgoing document can not be added without select the correspondence site type with main and sub .

In case add an outgoing document without content , the document will be saved in prepare screen    

 In case you dont fill one of mandatory fields in document properties , the fields appears in red color

In case you do not enter any of data , it will appear this sign

  In case you enter all the data , it will appears this sign         

  User can search for a sub sites in the simple search of the correspondence sites from the search results when select the site type and the main site .

 In case you add an electronic outgoing document user can save it as draft without sending it to Review screen when click on Save as draft button
