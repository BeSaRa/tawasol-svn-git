Attachment Types

Attachment Type is a classification statement to determine the types of attachments added within the outgoing , incoming, internal document , can use Attachment Types list durring add new attachment inside any document .

Attachment Types Screen content :

Arabic name .

English name .

Status of Attachment Types Active or Deactive .

Document Class (Outgoing , incoming, internal) .

item order belong to Attachment Types .

Edit Attachment Type from Edit Action .    

Delete Attachment type from Delet action.

The Attachment Types Screen consists of several options :

Reloading : Reload and upload the saved data in the screen .

Search :Search for any Attachment type Appears in the Attachment types screen using letter or number for the attachment type . name .

Sort : you can perform an ascending or descending order of data .

Add : Add new Attachment type .

Data should be used to add new attachment type:

Arabic name .

English name .

Document Class (outgoing , incoming , internal) .

Item Order .

Status Active or deactive .
