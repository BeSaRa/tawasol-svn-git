Review - Outgoing Mail

  The system allows to you to review the outgoing document which have been entered in you administration and take the action based on the authenticity of the enetered documents .

  When click on Review option , the system opens the Review screen to display all the outgoing documents electronic and paper .

  Review content screen :

  Reloading : reload and upload the saved data in Review screen  

  Search : search for any document Saved in the review screen from the search bar

  Sort : perform an ascending or descending sort of data

  Navigate between pages : you can navigate between pages through [ Next - Previous] 

  There are a list of options that contains the following actions in accordance to the permission granted to the user :

  Accept : accept the selected document to send it directly to Ready to send screen 

  Accept and Launch Distribution Workflow :  accept and launch the document to the user which will be selected from Send screen

  Edit content : edit the document content or prepare it when the document is electronic but when the document is paper you can edit the content by upload another one or by selecting Scanner

  Edit properties : the ability to edit the properities of the document which have been added when add the document

  Reject : Reject the outgoing document with written the reject reason and the document will appears in Rejected screen 

  Remove : remove the dcument from Review screen

  Notes :

The paper outgoing document can be exported directly to Ready to export screen when select "Send to export" action

The electronic outgoing document can be edited from you desktop when select Edit in Dekstop action then enter the server URL from settings screen
