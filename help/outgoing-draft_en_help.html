Draft - Outgoing Mail

The system allows you when add an electronic outgoing document to save it as draft in case you have not add all the document data without sending it to the Review screen and you can perfome all the operations of the document in a next time

Draft screen content :

Reloading : reload and upload the saved data in Draft screen

Search : search for any document appears in Draft screen by entering a letter or number for the document you  want to search for it from the search bar

Sort : you can perform and ascending or descending sort of data

The list of options which the user can take any of the following data in accordance to the permission granted to him :

 Send to review the document  when select "Send To Review" action from the step processor

Launch Distribution Worfklow and select only one user from the  Send screen to approve the document from his user inbox

Edit content by selecting one of three methods [Prepare document , Upload content , Edit content] when select "Edit" action then select "Content"     

Edit properties of the document which have been entered when add the outgoing document when select "Edit" action then select "Properties"      

Remove the document from Draft screen when select "Remove" action

Manage the document and the ability to add [ Tags , Comments , Linked Documents , Attachments , linked entities , Destinations] when select "Manage" action

View tracking sheet and display some histories of the document and all the procedures that have already occurred to the outgoing document when select "View Tracking Sheet" and select the history to be viewed 

Edit in desktop for the document and the document content will be opened on the desktop of the user when select "Edit in desktop" action

View sepecifc version for the document from Correspodence Versions screen and the ability to open any of version displayed to view the document content

Duplicate current version and the ability to select the dulicate options when select "Duplicate specific version" action

Duplicate specific version and the ability to select the document which user need to duplicate it then click on "Duplicate" icon then select the duplicte options  when click on "Duplicate specific version"

Notes :

You navigate between pages in Draft screen through [Previous - Next]

You should enter the server URL in the settings from the popup window  when click on Edit in Desktop
