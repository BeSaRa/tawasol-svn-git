Simple Incoming Add

  Tawasol system enables to users to add a simple incoming add from Incoming Mail

  User should fill the mandatory fields in Document Properties in Incoming Mail screen - Add new one

    Notes :

  User can not add a simple incoming document without select the correspondence site type with main and sub site

  In case you add a simple incoming document without content , the document will be saved in Scan screen

  User can not add incoming mail from simple add  if one of mandatory fields left as blank
