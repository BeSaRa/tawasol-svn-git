Simple add - Outgoing Mail

Tawasol system enables to users to add a simple outgoing document electronic or paper from Simple add

When add an outgoing document electronic or paper user should fill all the mandatory fields .

Notes :

User can not add an outgoing document from Simple add  without select the correspondence site type with main and sub site .

In case you add a simple outgoing document without content , the documet will be saved in Prepare screen

In case you add an electronic outgoing document from Simple add you can save it as draft when click on "Save As Draft" button
