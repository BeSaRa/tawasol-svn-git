Ready To Send -Outgoing Mail

  Ready To Send screen displays the outgoing document that have been reviewed and accepted from Review screen to appears in Ready to Send screen .

  Ready To Send screen content :

   Reloading : reload and upload the saved data in Ready To Send screen 

   Search : search for any documents saved in the Ready to send screen from the search bar 

   Sort : perform an ascending or descending sort of data

   Navigate between pages : you can navigate between pages through [ Next - Previous] 

  There are a list of options that contains the following actions in accordance to the permission granted to the user :

   Launch Distribution Workflow : the document sent directly to the user which have been selected from Send screen

  Send To Export : send only the paper documents to Ready To Export screen 

  Broadcast : broadcast the paper outgoing document only to the organization units and workflow groups that are selected from broadcast screen 

  Edit content : edit the document content or prepare it when the document is electronic but when the document is paper you can edit the content by upload another content or by selecting the scanner

  Edit properties : the ability to edit the properties of the document which have been added when the document

  Open or preview the document : you can open the document and view all the saved data when select "Open" or "Preview" action

  View specific version : the ability to view specific versions for the selected document in correspondence versions screen and when user edit the content of the document a new version will appear 

  Duplicate specific version : the ability to duplicate both paper and electronic outgoing document with the selection of duplicate options 

  Edit in desktop : edit the content of any electronic document from your desktop when select "Edit in desktop" action then enter the server URL from the settings screen

  Duplicate current version : user can duplicate only electronic document
