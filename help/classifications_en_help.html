Classification Page

TAWASOL has two levels of Classification:

  1. Main Classification
  2. Sub Classification

 
To add a new Classification, User clicks on Add Classification button and the system open add new main classification page with the followings parameters:

1. Basic Properties (TAB):

  • Arabic Name (Mandatory)
  • English Name (Mandatory)
  • Status (Active, Inactive)
  • Is Global Main Classification. In this case, the system will assign the main Classification to all organization units and disable the Organization Units TAB.

2. Organization Units (TAB): you can manage the organization units that can use the new main Classification. by doing the following actions:

  • Add
  • Remove

you can also give the permission to Organization Unit and all sub units in the same time.

3. Sub Classification (TAB): User manages the related sub Classification. User can do the following actions:

  • Add New
  • Edit
  • Remove
  • Open

To Edit an existing main classification, the admin clicks on Edit button and he/she can edit the followings properties
 

To delete a Classification, admin clicks on the delete button  and the system will display a confirmation message before deleting . admin will always be able to remove the Classification in case not linked to any document.

To Activate/Deactivate a Classification the admin shift the status button right to activate or left to deactivate the required type. The admin can select and activate/deactivate multi classifications at the same time.
