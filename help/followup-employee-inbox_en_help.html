Followup Employee's Inbox

  Followup Employee's inbox is followed up at two levels at the level of managment and the sections of the managment at the same level in the same registery of organization units when select followup employee's inbox.

  Through the followup employee's inbox screen you can review the employee's inbox in the order in their mail.

  Followup employee's inbox content :

  Reloading : reload and upload the saved data in the screen

  Search : search for the document exist in the followup employee's inbox screen from the search bar.

  Sort : you can perform an ascending or descending sort of data.

  Navigation between pages : user can navigate between pages through [Next -Previous]

  User can take the following specific actions through the list of options of the selected document :

  Tansfer to another  employee : transfer the document to another employee in his user inbox and identify the following fields :

  Organzation unit from the drop-down list

  Select user from the drop-down list 

  The reason then click on "Transfer" button to transfer the document to the user inbox of the selected user 

  View tracking sheet : which contains a number of records and reports that have occured to the document

  Open the document : open the document and view all information related to the document when select "Open" or "Preview" action and the ability to take the following actions :

  Add to favorite

  Download the main document

  Send the document by Email 

  Get Link

  View specific version

  Duplicate specific version

  Duplicate current version for the eletronic documents only

  Transfer to another employee

  Document information : the system displays the following information which have been entered during the addition of the  document [Security level , Priority level , tags , author , comments]

  Notes :

  When select followup employee's inbox from Inbox  a screen will appears to select the organization unit and the application user then click on "Get Inbox" button

  You can change the Organization unit and select another user when click on this icon exist next to the search bar
