Ready To Send -Internal Mail

Ready To Send screen displays the internal document that have been reviewed and accepted fromReview screen to appears in Ready To Send screen

Ready to send screen contains the following  procedures which user can take any of them in accordance to the permission granted to him :

Launch Distribution Workflow   : the document sent directly to the user which have been selected from send screen

Broadcast : broadcast the paper internal document only from Ready to send screen to multiple organizations and groups from broadcast screen

Edit  : user can edit content or properties for paper and electronic internal document

View specific version : the ability to view specific version for the selected document in correspondence versions screen and when user edit the document's content a new version will appears in versions screen

Duplicate specific version : duplicate paper and electronic document with the ability to select the document version which user need to duplicate it

Duplicate current version : the ability to duplicate only the eletronic document

Notes :

User can edit content from inside the electronic document when open the document then click on "Edit Content" button 

User can navigate between pages through [Next - Previous]
