Group Mail

Tawasol system enables users to send the document to the departments that do not have correspondence registery

Onyl the electronic approved document and paper are sent to the group mail in Inbox queue

The document will be send to the Group mail screen through the send screen when select a non registery deparmtent then click on "Send to whole department" icon

                                              

Group Mail screen contains the following procedures which user can take it in accordance to the permissions granted to him :

Forward : the ability to forward the document to users and organizations

Terminate : terminate the document with the reason for termination

  Broadcast : broadcast the document to organization units and groups

Open or preview document : open the document's content and view all the information related to the document

  View specific version

Duplicate specific version

Get link

Manage document  : the ability to manage document and add some data to the document like [ tags , comments , attachments , linked documents , linked entities]

View tracking sheet  : which contains a number of records and reports that have been occured to the document

Download the main document

Send the document by Email : Send the document by Email by  two methods [ Send the document only or send the  compsoite document  as attachments including attachments and linked documents]

Notes :

The document will be send to all users in the department when  select  send to whole department

There are some actions user can take it when click on "Quick action" icon
