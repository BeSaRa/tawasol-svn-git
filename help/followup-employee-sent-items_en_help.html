Follow-up Module

If user has the first privilege which is (personal follow-up) a new queue for personal follow-up will appear

He can assign follow-up for document starting from review queues, inbox and all search screens.

Action follow-up can be taken from outside also inside the document.

At assign follow-up screen mandatory field for date will be added to select the follow-up date.

For outgoing/incoming documents have follow up date, system will show the document follow up date to be the default date and user can change it to any other date.

For multi correspondence outgoing documents have follow up date , system will show the maximum follow up date to be the default date and user can change it to any other date.

For outgoing/Incoming documents that do not have follow-up , user will be able to add documents for his follow-up with any date and will show by default the service level agreement set value.

For Internal documents, user will be able to add documents for his follow-up with any date and will show by default the service level agreement set value.

For outgoing/incoming documents with a passed follow up date, user should be able to add them for follow up with any date.
