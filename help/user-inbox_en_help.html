User Inbox

User Inbox screen it's one of Inbox list screens , allows you to follow up your personal mail and follow up all actions already taken in all documents you have .

User Inbox content :

Serial Number for every document .

Subject of the document .

Receive date the date of the receipt of the documents in the sent items .

Actions taken on any document when sent or creat document .

Sender the user sent this document .

Due Date of the document .

Correspondence site the sites which the documents have been sent to .

Tawasol System allows you to view User inbox as :

Grid View .          

Magazine View .

User can change view options when click on display icon in user inbox screen .

Tawasol allows you to display a thumbnail of the content of the document by clicking on the side of the document , only can showen in magazine view .

User Inbox screen consists of several options :

Reloading : reload and update the saved data in the screen .

Search : Search for any document appears in user inbox screen using aletter or a number for the document .

Sort : you can perform an ascending or descending order of data .

The list of actions can taken in user inbox screen :

Document info it's Data was entern when add document .

Open the document and view all the general data of the document and the procedures by pressing on "open" action .

Terminate document and write the reson , can find it again when search on it from "Terminate" action .

Get Link makes you open the document outside the system by press on the link display when take "Get Link" action .

Add To can add any document to favorite or folder .

Send the document by mail the main document or composite document with attachments and linked documents .

Edit in desktop in case document not approved can open it in desktop office and edit the content .

Duplicate speific verision of any copy of version you want to duplicate and update on it .

Edit Content in case the document is electronic user can ( prepare document or upload content or edit content ) in case paper document user can ( upload content or scanner ) .

Edit properties can update some data in document properties from "Edit properties" action .

View Tracking Sheet wit all history of the document and can export or print this history .

Approve any electronic document haven't approved yet in case user have this permission .

Manage documents can add or update ( Tags - comments - Linked documents - Attachments - Linked Entities ) .

Download document the main document or composite document with attaments and linked documents .

Forward any document to users and organization units .

Replay to any document and in this case system choose automatic the user who was sent this document to you , you can update it .

Create Replay in case the document was incomig only .

Subscribe any document and follow exactly the action you want to know .
