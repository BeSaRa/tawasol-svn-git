Draft - Internal Mail

  You can through tawasol system add an electronic internal document and save it as draft to be saved in Draft screen without sending it to Review screen .

  Draft screen contains some procedures which user can take it in accordance to the permission granted to him :

  Send To Review

  Launch Distribution Workflow   : launch the document to one user from Send screen in case the document is electronic to approved the document from his user inbox but in case it is paper it is send to more than one user

  Edit : the ability edit content or properties

  Remove : remove the document from draft screen

  Edit in desktop : edit in desktop the document's content of the electronic document from the desktop of the user

  View specific version        : view specific version from correspondence versions screen and the ability to open any of version displayed to view the document content

  Duplicate current version :  the ability to duplicate current version for the electronic document only with the selection of duplicate options

  Duplicate specific version : duplicate a specific document and the ability to select the document which user need to duplicate it then click on "duplicate" icon after that select the duplicate options
