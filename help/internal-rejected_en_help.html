Rejected screen - Internal Mail

Tawasol system enables to users from Rejected screen to view the documents that have been reviewed but still not take a decision  about their approval and have been rejected when select "Reject" action from Review screen

When reject the document from Review screen it will appears in Rejected screen

Rejected screen contains the following procedures which user can take any of them in accordance to the permissions granted to him :

Send To Review : send the rejected document to Review screen

Launch Distribution Workflow : the document can be send directly to the user which will be selected from Send screen

Archive : archive the rejected document

Edit  : the ability to edit content or properties of the selected document

Edit in desktop : edit the document's content of the electronic document only

View specific version : the ability to view a specific version of the document's content

Duplicate specific version : the ability to duplicate electronic and paper document with the selection of duplicate options

Duplicate current version : the ability to duplicate electronic document only

Notes :

User can edit document's content from inside the electronic document when open the document then click on "Edit content" button
