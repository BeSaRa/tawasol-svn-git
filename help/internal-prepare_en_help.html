Prepare -Internal Mail

  You can through Tawasol system to save the internal documents in Prepare screen when add internal document without content

  Prepare screen content :

  Reloading : reload and upload the saved data in Prepare screen

  Search : search for any document appears in Prepare screen from the search bar 

  Sort : you can perform an ascending or descending sort of data

  The list of options which user can take any of the following procedures in accordance to the permissions granted to him :

  Prepare content  : when create content for the document it will be moved directly to the Review screen

  Edit document properties : edit the properties of the internal document which have been entered when add the document

  Remove  : remove the selected document from Prepare screen

  Document Information : the system displays the information which have been entered when create the document
