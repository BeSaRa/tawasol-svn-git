G2G Incoming

Government incoming: Displays all incoming documents received from other government entities by

       government correspondence

 

 

 

To reload received document click on: 

Inbox displays all incoming documents received by the department via government correspondence,

      as per the window:

 

 

Description of each field in the Incoming inbox is given below:


Subject: Subject of the incoming document


Document Type: Defines the type of incoming document as Original or Copy.

Security Level: Shows the Security Level of the incoming document as Normal or Classified as defined

       by the Sender.


Document number: This is an optional field and displays the document number if entered from the sender


Government Reference number: This is the unique Government Correspondence reference number

       that is created when document is exported via Government correspondence. This number will be used for

       referencing or tracking the document for any purpose.


Receiving Date: Displays the date and time the document was received.


Sender department: Displays the sender Government entity and Organizational Unit details.


User can open incoming document, view attachments, linked Documents and person details related to the

       incoming document.
