Rejected screen - Incoming Mail

Tawasol system enables to users  from Rejected screen to view the document that have been reviewed but still not take a decision yet for their approval action and have been rejected when select Reject action from Review screen

The rejected document appears which has been rejected from Review screen appears in Rejected screen

User can take any of the following actions :

Send To Review : send the document to review screen when select "Send To Review" action 

Launch Distribution Workflow : the document can be send directly to one user or more than one or to organization units from send screen 

Open or preview document : the ability to open the rejected document and view all the data when click on "Open" or "Preview"               

                                                               

 Document Information : the system displays all the information that have been entered when add the document when click on  "Document Information" from the step processor                                      

Archive : Archive the rejected documents 

Notes :

You search for any document in the Rejected screen from the search bar 

You navigate between pages in the rejected screen through [ Next - Previous]

There are some actions from the step processor in the Rejected screen can appear or disappear in accordance to the permission granted to the user
