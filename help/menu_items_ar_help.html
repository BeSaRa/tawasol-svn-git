قوائم النظام

 قوائم النظام هى جزء ثالث بين نظام تواصل و قاعدة البيانات لدمج و استدعاء الميزة مثل[ICN , Reports] من خلال العنوان الذى يتم إضافته و عرضه على نظام تواصل عند إضافة قائمة نظام جديدة.

    تنقسم قائمة النظام إلى عنصر رئيسى و فرعى، ويتم إضافة قائمة نظام فرعى بعد إضافة قائمة النظام الرئيسى.

محتوى شاشة قوائم النظام :

 إضافة قائمة نظام جديدة.

إعادة تحميل وتحديث شاشة قوائم النظام.

 شريط البحث والذي يمكن من خلاله البحث عن أي قائمة نظام موجودة بشاشة قوائم النظام:

      الحالة : فعال ليتم عرضها فى نظام تواصل.

الحالة : غير فعال ليتم إخفائها من النظام.

  عند إضافة قائمة نظام جديدة يجب ملء الحقول الإجبارية التالية:

    الاسم باللغة العربية : كتابة اسم قائمة النظام باللغة العربية.

الاسم باللغة الإنجليزية : كتابة اسم قائمة النظام باللغة الإنجليزية.

  ترتيب العنصر : إدخال رقم الترتيب لقائمة النظام.
