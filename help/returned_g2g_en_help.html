G2G Returned

Displays outgoing documents that were returned back via Government Correspondence system. Details of

       returned document include Subject, Date, Type of Document and Security Level, Document number, Government Reference number and Sender department.

 

 

Click on comment box near the subject to see the comments and reason for return of the document.

 

User can open and review the document and do any of the below actions:

 

Resend document : Send document again via government correspondence

Edit document: There will be a confirmation message before editing the document.

      The outgoing document has to be exported again after edit.

Remove transaction: Document will be removed from g2g returned queue.
