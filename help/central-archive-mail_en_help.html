Central Archive Mail

  Central archive mail contains the documents that will be exported to external sites or to G2G through Ready To Export screen

  When add an outgoing document and export it to an external sites , it is first exported to the department inbox in Ready To Export screen then export to central archive mail when select "Send To Central Archive" action

  Ready to Send screen in Central Archive Mail contains the following procedures :

  Export : when select export action a screen will opened to select the export type [export by grouping , export by selection] then click on Export button for the electronic document or click on "Print and terminate" for the paper document

  Return : the document will be returned to Ready to Export screen of the Department inbox in which the document was added with the return reason

  Print barcode

  Terminate the document

  Manage the document and add some of data to it like [ tags , comments , attachments , linked documents , linked entities]

  Download the main document

  View tracking sheet

  Open or preview the document to view the document's content and all its information

  View specific version

  Duplicate specific version

  Duplicate current version for the electronic documents only
