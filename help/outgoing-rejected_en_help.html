Rejected screen - Outgoing Mail

  Tawasol system  enables to users  from Rejected screen to view the document that have been reviewed but still not take a decision yet about their approval and have been rejected when click on Reject action from Review screen 

  The rejected documents appear in the rejected screen 

  User can take any of the following actions in the Rejected screen from the step processor :

  Send To Review : the rejected document  send to the review screen when click on Send To Review action

  Launch Distribution Workflow : the rejected document can be send directly to the user which has been selected from Send screen 

  Open or Preview document : the ability to open the rejected document and view all the general data when click on "Open" or "Preview"

                                                                                                     

  Document Information : the system displays all the information which have been entered when add the document when click on document information 

  Archive : archive the rejected document

  Edit in desktop : edit only the electronic outgoing document from Edit in Desktop action and the document content will opened in a word file from your dekstop

Notes :

You should enter the server URL in the settings from the popup window when click on Edit in desktop

                                                                                          

There are some actions appear or disappear in Rejected screen in accordance with the permissions granted to the user
