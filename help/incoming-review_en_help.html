Review Incoming

   The system allows to you to review the incoming document which have been entered in your administration .

  When click on Review option from Incoming Mail queue the system opens the Review screen which shows the incoming documents that have been added .

  Review screen content :

   Subject : subject of the document

   Priority level : Normal , Urgent , Top Urgent

    Security Level : Normal , Confidential , Private

    Main-Site : which have been selected when add the document

  Sub-Site  : which have been selected when add the document after select the main site

   Created On : the date added in the incoming document

  There are some actions and orders that user can take during the review process :

   Reloading : reload and upload the saved data in the review screen

 Filter : search for any document appears in the review screen using a letter for the document you want to search for it in the search bar

  Sort : you can perform an ascending and descending sort of data

  The list of options which the user can take any of the following procedures in accordance with the permission granted to him :

  Accept : the document is accepted and send directly to Ready To Send screen

  Reject : the document is rejected and send to Reject screen                              

  Accept and Launch Distribution Workflow : accept and send directly to the users which have been select it from Send screen

  Print Barcode : print the barcode of the incoming document    

  Remove : remove the selected document from Review screen 

  Edit : Edit content of the document by uploading another content or by scanner and also you can edit the properties of the document        

  View Specific Version : view specific version of the correspondence versions of the incoming document documents                                                 

   Duplicate Specific Version : duplicate the document with the selection of the duplicate options [Related Books , Related Objects , Attachments , tags , Correspondence Sites]                                                                                                                       

  Open : open the document and view all the general data of the document and the procedures

   Preview : preview the document and view all the data of the document
