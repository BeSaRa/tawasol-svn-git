Add Incoming

The system enables to users to add incoming mail .

First , user should fill the mandatory fields in document properties .

The following data user can manage it when add an incoming document :

Add tags from Manage Documents Tags 

Add linked Documents by searching for any documents in search for document screen from manage linked documents

Add attachments by import the attachment or by Dragd & Drop from Manage Document Attachments

Add comments from Manage Document Comments    

 Add external employee or new entity from Manage Linked Entities

 Notes :

The incoming documents can not be added without select the correspondence site type with main and sub site

 In case user add an  incoming document without prepare document content , the content will be saved in Scan screen

 User can not add an incoming mail if one of mandatory fields left as blank

User can add attachments by scanner when select scanner from Manage Document Attachments

 User can add the correspondence site type with  the main and  the sub site by one of two methods : Simple search or Advanced search and this is determined from Global Settings
