Approved

The Approved screen displays the internal document which have been approved from User Inbox screen

Approved screen contains some of procedures that user can take in accordance to the permission granted to him :

Edit After Approved : the ability to edit the document after it has been approved and will be transferred to the Review screen after modifying the properties of the document and all its contents when select "Edit After Approved"

Broadcast : the system enables to you to broadcast the document to one organization unit or more than one and also can broadcast to workflow groups

Launch Distribution Workflow : send the approved internal document to one or many users from Send screen

Add to favorite : Add the document to favorite documents screen

Open or Preview document : the ability to open the document and view its content and all the saved data

View specific version      : the ability to view specific version for the document and display the document's content that have been selected from from correspondence versions screen

Duplicate specific version : duplicate the selected document from correspondence versions screen with the selection of duplicate options

Duplicate current version   : dupplicate the document with the selection of duplicate options

Terminate : terminate the document with writing the terminate reason
