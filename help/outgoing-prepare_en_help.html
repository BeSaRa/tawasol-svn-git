Prepare - Outgoing Mail

Tawasol system enables to save the outgoing document in Prepare screen when add an outgoing document without content

Prepare Outgoing Content :

  Reloading : reload and upload the saved data in the Prepare screen

Search : search for any document appears in the Prepare screen from the search bar

Sort : you can perform an ascending or descending sort of data

The list of options which the user can take any of the following procedures in accordance to the permissions granted to him :

Create content : when create content for the document it will be moved directly to the Review screen

Edit document properties : edit the properties of the outgoing document which have been entered when add the document

Remove : remove the selected document from prepare screen when select "Remove" action from the step processor
