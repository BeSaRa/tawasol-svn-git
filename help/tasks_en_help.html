my tasks

System allows the user to add new tasks from the task manager list by entering the necessary information such as
( task title- link document - Participants, if any )

Task title - start date – Due date – Participants (If there is) – Linked document.

Optional information as task description

Also can defined if user will work in the task all the day or defined hours

Can add new task from the below options

From + tab in tasks screen

Can drag the dates from calendar and add new task

From outside document in user inbox can choose add task and in this case this document will be linked in the task

If launch any document content due date the new task will create directly and the receiver user will be participant in this task

After add new task user will have notficatin in task notfication list

Task owner will be apple to send remined to all particapant by mail notfication

User can specify the duration of the task and the Add Task screen appears with the same predefined time

This can be done directly from personal mail if the task is related to a particular document.

Can display taks calender by ( date – week – month ) or by task status ( cancelled – peneding – in progrees – complated )
