my-Follow-up

Once user assign follow-up for a document it will be added to personal follow-up queue.

Personal follow-up will be separated by subfolders.

User can either navigate to a folder to see its items or he can choose to show all items regardless of the folders and the system
will show the maximum number of items as per the preferences.

The follow up screen will show by default active follow up items however, the user can still show the terminated follow up items on monthly basis.

Folders will contain counters.

Once user assign follow-up for the first-time system will automatically create public folder for the user under name public.

Folders contain follow-up items can’t be removed until the follow-up items are moved to another folder
