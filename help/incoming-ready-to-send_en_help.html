Ready To Send - Incoming Mail

  Ready To Send screen displays the incoming documents that have been reviewed and accepted from review screen to appears in Ready To Send screen .

  Ready To Send screen content :

  Subject : the subject of the document

   Priority level : Normal , Urgent , Top Urgent

   Security level : Normal , Confidential , Private

  Creator : who creates the incoming documents

  Created On : creation date of the document

  There are some actions and order users can take it in Ready To Send screen :

  Reloading : reload and upload the saved data in Ready To Send screen

  Search : search for any document appears in ready to send screen using a letter for the document that you want to search for it in the search bar 

  Sort : you can perform an ascending or descending sort of data

The list of options which the user can take any of the following procedures in accordance to the permissions granted to him :

  Launch Distribution Workflow : the document sent directly to the users which have been selected from Send screen

 Archive : archiving the incoming document

 Broadcast : broadcast the incoming documents to the organization units and workflow groups that are selected from broadcast screen

Edit : edit content of the document by uploading another content or by scanner also you can edit the properties of the document 

View Specific version : the ability to view specific versions for the correspondence versions of the incoming documents      

Duplicate specific version : duplicate the document with the selection of duplicate options [ Related Books , Related Objects , Attachments , Tags , Correspondences sites ]     

 Open or Preview document : user can open the document and view all the general data of the document when click on "open" or "preview"

                                                                         

   Notes :

     User can navigate between pages through [Next - Previous]

     User can select multiple documents and take some actions on them when click on muti select icon then the number of selected documents will appear 

   User can takes some actions when select open or preview action from the step processor then click on the three dots   inside the document
