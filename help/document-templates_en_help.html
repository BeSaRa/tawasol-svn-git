Document Templates

Document Template used to saved templates you can used after that when add new electronic document outgoing or internal and edit on it easily .

Document Templates screen content :

Arabic name .

English name .

Document type (Outgoing , Internal , Both) .

Status of documnt template active or deactive .

Edit document template from edit action .        

Delete document template from delete action .

The Document Template screen consist of several options :

Reloading : Reload and upload saved data in the screen .

Search : Search for any document template appears in the document templates screen using letter or number for the document template name .

Sort : you can perform an ascending or descending order of data .

Organization Unite : to show the private template inside the organization .

Add : add new document template .

Data should be used to add new document template

Arabic name .

English name .

Document type (Outgoing , Internai , Both) .

Signatures count needed to approve the document .

Upload document from your Personal computer .

Status active or deactive .
